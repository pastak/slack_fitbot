# slack_fitbot

This is SlackBot tells us Infomation via Fitbit

## Install

- Install [Bot](https://slack.com/apps/A0F7YS25R-bots) to your team
- `$ mv config.sample.js config.js` and fill in.
- `$ npm install`
- `$ npm run start`
